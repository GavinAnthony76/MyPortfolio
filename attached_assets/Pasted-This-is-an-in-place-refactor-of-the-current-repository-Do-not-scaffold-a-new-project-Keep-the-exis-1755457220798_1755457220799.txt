2) Add a minimal diagnostics script (no framework changes)

Create scripts/object-diagnose.ts and add npm script "diag:objects".
The script should accept a CLI arg --url <full_image_url> and print:

Status code, Content-Length, Content-Type for HEAD

Bytes received for GET

Status/bytes for a Range: bytes=0-0 request
(Use Node 18+ fetch; no dependencies.)

// scripts/object-diagnose.ts
const url = process.argv[process.argv.indexOf("--url")+1];
if (!url) { console.error("Usage: npm run diag:objects -- --url https://..."); process.exit(1); }
const head = await fetch(url, { method: "HEAD" });
console.log("HEAD", head.status, head.headers.get("content-length"), head.headers.get("content-type"));
const getRes = await fetch(url);
const getBuf = Buffer.from(await getRes.arrayBuffer());
console.log("GET", getRes.status, "bytes:", getBuf.length);
const r = await fetch(url, { headers: { Range: "bytes=0-0" }});
const rBuf = Buffer.from(await r.arrayBuffer());
console.log("RANGE", r.status, "bytes:", rBuf.length);


In package.json add:

"scripts": {
  "diag:objects": "node --experimental-modules --no-warnings scripts/object-diagnose.ts"
}


(Adjust if using tsx; otherwise keep as pure .ts compiled to .js.)

3) Fix the proxy to stream correctly

Wherever we proxy from object storage, ensure we stream the upstream body and forward sane headers. Replace any res.send(...)/manual Content-Length with a streaming pipe. Also pass through Range for partial content.

Express/Node (Common Case):

// Example pattern; apply to the actual route file found in Step 1
app.get("/objects/portfolio/:key", async (req, res) => {
  const key = req.params.key;
  const upstream = await fetch(`${process.env.OBJECT_ORIGIN}/objects/portfolio/${encodeURIComponent(key)}`, {
    headers: req.headers.range ? { Range: String(req.headers.range) } : {}
  });

  res.status(upstream.status);

  upstream.headers.forEach((value, name) => {
    const n = name.toLowerCase();
    if (!["transfer-encoding","content-encoding","connection"].includes(n)) {
      res.setHeader(name, value);
    }
  });

  const ct = upstream.headers.get("content-type");
  if (ct) res.setHeader("Content-Type", ct);
  const cl = upstream.headers.get("content-length");
  if (cl) res.setHeader("Content-Length", cl);

  if (!upstream.body) { res.end(); return; }

  // Node 18+ Readable from web stream:
  (upstream.body as any).pipe(res);
});


Next.js (Node runtime API route):

// pages/api/objects/[key].ts (or app/api/.../route.ts with Node runtime)
import type { NextApiRequest, NextApiResponse } from "next";
import { Readable } from "node:stream";

export default async function handler(req: NextApiRequest, res: NextApiResponse) {
  const { key } = req.query as { key: string };
  const upstream = await fetch(`${process.env.OBJECT_ORIGIN}/objects/portfolio/${encodeURIComponent(key)}`, {
    headers: req.headers.range ? { Range: String(req.headers.range) } : {}
  });

  res.status(upstream.status);
  upstream.headers.forEach((v, n) => {
    const nn = n.toLowerCase();
    if (!["transfer-encoding","content-encoding","connection"].includes(nn)) res.setHeader(n, v);
  });

  const body = upstream.body;
  if (!body) return res.end();
  Readable.fromWeb(body as any).pipe(res);
}


Rules:

Do not manually set Content-Length unless it comes from upstream.

Always pass through Content-Type.

Never send a placeholder string or res.end(" ").
