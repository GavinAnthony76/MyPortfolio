## Environment Variables Required

Create these environment variables in your Replit deployment:

```bash
# Database
DATABASE_URL=your_production_database_url

# Security
SESSION_SECRET=your_secure_random_session_secret
NODE_ENV=production

# Optional PayPal Integration
PAYPAL_CLIENT_ID=your_paypal_client_id
PAYPAL_CLIENT_SECRET=your_paypal_client_secret

# Server
PORT=5000
```

## Custom Domain Configuration

1. **In Replit Deployments:**
   - Navigate to your deployment settings
   - Add custom domain: `www.gavineanthony.com`
   - Configure SSL/TLS certificate (auto-generated by Replit)

2. **DNS Configuration:**
   ```
   CNAME: www.gavineanthony.com -> your-replit-app.replit.app
   ```

## Database Setup

1. **Production Database:**
   - Use the provided DATABASE_URL from Replit's PostgreSQL addon
   - Run migrations: `npm run db:push`

2. **Admin User Setup:**
   - Admin user is auto-created on first deployment
   - Default credentials are set via environment variables
   - Change default password immediately after deployment

## Support and Troubleshooting

### Common Issues
1. **Database Connection Issues**: Verify DATABASE_URL is correct
2. **Session Issues**: Check SESSION_SECRET is set and secure
3. **Image Loading Problems**: Verify asset paths and storage configuration
4. **Rate Limiting**: Adjust limits in server/index.ts if needed

### Emergency Procedures
- **Rollback**: Use Replit's deployment history to revert
- **Database Issues**: Check logs and verify connection strings
- **Security Incidents**: Enable maintenance mode, investigate, and patch
